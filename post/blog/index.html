4.6. 总结

    状态查看 
    git status 查看工作区、暂存区状态
    添加 
    git add [filename] 将工作区的“新建/修改”添加到暂存区
    提交 
    git commit -m"commit message" [filename]  将暂存区的内容提交到本地库
